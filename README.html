Overview & Benefits

SOHMNAMP is built on the Web Audio API and Three.js, delivered inside an Electron desktop app. Version 4.0.4 ships as a signed macOS DMG and a Windows x64 setup EXE, with local playback, live streams, podcasts, native video playback, and a visualizer-first interface. Your local files stay on your machine; network access is used only when you browse radio stations, podcasts, or external stream URLs.

⚡
Zero Setup
Open the app, drop audio or video, browse stations, or import a stream URL. No account or cloud library required.
🔒
Local Processing
Playback, EQ, reverb, file analysis, recording, playlist bundles, and visualizer rendering happen locally in the app.
🎛
Producer-Grade Tools
10-band EQ, algorithmic convolution reverb, BPM detection, key detection, crossfade, repeat modes, and real-time 3D visuals.

100,000+ Radio Stations

Audio Signal Chain

Every node runs natively in the Web Audio API. The analyser sits before the EQ so it captures the raw signal for visualizer reactivity. The EQ shapes before reverb, meaning the reverb tail is also EQ'd.

Source→ Analyser→ 10-Band EQ→ Convolution Reverb→ Gain Node→ Output+ Recording Tap

The Recording Tap branches from the master gain node into a MediaStreamDestination, capturing exactly what you hear — EQ, reverb, and all.

Voice Control New

Powered by Whisper running on-device via a persistent Python subprocess. Activate with 🎙 VOICE or the global shortcut ⌘⇧V / Ctrl⇧V. If ANTHROPIC_API_KEY is present, Claude Haiku can parse richer natural-language commands before the regex fallback runs.

How it works: Voice mode pauses and mutes app audio, opens the selected microphone, watches for speech with a VAD loop, records the utterance, converts it to 16 kHz mono WAV, pipes it to the warm Whisper process, then restores playback after the command is handled.

CommandExamplesAction
Play / Pause"play", "pause", "stop"Toggles or stops playback
Navigation"next", "previous", "skip"Moves to next or previous track
Volume"volume up", "louder", "quieter", "mute"Adjusts master gain
Search"play [artist/title]"With Claude intent parsing, searches the playlist and jumps to a matching track
Shuffle"shuffle on", "shuffle off"Toggles shuffle mode
Repeat"repeat on", "repeat off"Toggles repeat-track mode
Seek"skip forward", "skip back"Moves the current track by 10 seconds
Fullscreen"fullscreen", "windowed"Enters or exits fullscreen visualizer mode
Visualizer"nebula", "aurora", "galaxy"Switches to named visualizer mode
Voice"voice off", "stop listening"Stops the microphone listener

The Whisper server stays warm between commands for faster response. Voice and gesture controls are hidden in Mac App Store builds where camera and microphone access are disabled.

Reverb Engine

A convolution reverb built entirely in the Web Audio API. The impulse response is generated algorithmically with Gaussian early reflections and a low-pass filtered noise tail — no sample files required. Toggle with B.

Room TypePre-delayDampingCharacter
Room10 ms0.55Natural, intimate space — subtle and transparent
Hall25 ms0.35Large concert hall — long decay with early reflections
Chamber6 ms0.60Studio echo chamber — tight, present, controlled
Cave45 ms0.20Deep cavernous decay — long and dramatic
Spring2 ms0.70Classic spring reverb character — vintage warmth
Plate3 ms0.65Studio plate — bright, smooth, professional

Three controls: Room Size (0–1) · Wet Mix (0–40%) · Decay (0.1–4.0 s). The impulse response rebuilds in real-time as you adjust.

Technical Architecture

Everything user-facing runs in the browser layer of Electron. The app uses a small preload bridge and main-process IPC for native window controls, dedicated video windows, permissions, podcast RSS fetching, update events, and the persistent speech subprocess in non-MAS builds.
